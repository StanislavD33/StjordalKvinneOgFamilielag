# Kvinne og Familielag

Website created for the "Kvinne og Familielag" organization based in Stjørdal, Norway.

## Features

- React + Vite setup
- SEO optimization with React Helmet
- Responsive design
- Integration with Headless CMS (Strapi)

## Tech Stack

- React
- Vite
- JavaScript
- Tailwind CSS
- React Router
- React Helmet
- Strapi (Headless CMS)

## Installation

Clone the repository
git clone https://github.com/StanislavD33/StjordalKvinneOgFamilielag.git

Go to the project folder
cd kvinne-og-familielag

Install dependencies
npm install

Run development server
npm run dev

## Build

To create a production build:
npm run build
